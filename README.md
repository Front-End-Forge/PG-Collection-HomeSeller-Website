# 👗 PG Collection Studio - Tailoring & Masterclass Booking Platform

## 🌟 Overview
PG Collection Studio is a premium, localized React-based web application tailored for a custom home tailoring and Aari embroidery business in Salem. The platform features an ultra-modern, fully responsive interface built specifically with a **"Tamil-First"** approach to ensure maximum accessibility, readability, and conversion among local demographics.

## ✨ Key Features
- **High-Conversion Booking Flows**: Custom luxury booking forms for Designer Gowns and Aari Work.
- **Widescreen & Mobile Responsive**: Advanced `max-w-7xl` architecture ensures flawless symmetrical scaling on desktop monitors while providing a seamless, thumb-friendly mobile app experience.
- **Automated WhatsApp Checkout**: Direct integration with the studio's business line (`+91 7010006789`). The system intelligently captures the user's React UI states (Apparel Type, Embroidery Method, Delivery Selection) and translates them into a highly structured, auto-populated WhatsApp invoice in pure Tamil.
- **Aari Masterclass Portal**: A dedicated academy enrollment tab showcasing the syllabus, flat pricing (`₹3000`), offline/online batch toggles, and government certificate add-ons (+₹1000). 
- **Premium UX/UI**: Implements subtle gradients, custom animated dropdown widgets (bypassing native browser `<select>` limitations for long text), high-contrast accessibility standards (for 35+ age demographics), and large, legible typography.

## 🛠 Tech Stack
- **Frontend Framework**: React
- **Styling Engine**: Tailwind CSS (Utility-First Layouts)
- **Icons**: Lucide React
- **Checkout Integration**: Native WhatsApp URL Schema (`wa.me`)

## 📦 Project Structure
- `frontend/src/components/CustomGownPage.jsx` - The core Custom Tailoring Booking Engine.
- `frontend/src/components/AariClassPage.jsx` - The Academy Masterclass Enrollment Portal.
- `frontend/src/mockData.js` - Centralized mock backend datasets, portfolio arrays, and syllabus information.

## 🚀 Getting Started
1. Navigate to the `frontend` directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. The application will launch on your local host port (usually `http://localhost:3005`).
